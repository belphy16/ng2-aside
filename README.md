#ng2-aside
Imagine an aside menu that pushes your page or comes over it and so on.

### NPM

```
$ npm install ng2-aside

```


### Configuration

| Parameter        | Default                        |         |
| :-------------   |:-------------------------------| :-----  |
| side             | left                           |         |
| width            | 240px                          |         |
| is-backdrop      | false                          |         |
| sideMode         | over                           |         |
